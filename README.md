# CRUD de Alunos

Projeto simples de CRUD (Create, Read, Update, Delete) de alunos,
desenvolvido para estudo de React e consumo de API.

## Tecnologias
- React
- JavaScript
- Bootstrap
- MockAPI

## Funcionalidades
- Listar alunos
- Cadastrar aluno
- Editar aluno
- Remover aluno

## Como rodar o projeto
npm install
npm start
